# 记忆类型（Tulving 1972）

## 核心定义

Tulving 在 1972 年提出的记忆系统三分法：

| 类型 | 用途 | 特点 |
|------|------|------|
| **情境记忆**（Episodic） | 记录过去 | 我干了啥 |
| **语义记忆**（Semantic） | 提炼规律 | 我知道啥 |
| **程序化记忆**（Procedural） | 决定行动 | 我会做啥 |

## 循环过程

```
经历 → 知识 → 技能 → 新的经历
  ↓       ↓       ↓
情境   语义   程序化
记忆   记忆   记忆
```

## AI 记忆系统中的应用

- **情境记忆**: 对话历史、操作记录
- **语义记忆**: 提炼的知识、规则
- **程序化记忆**: 反馈、最佳实践
